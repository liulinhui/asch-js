# Asch javascript library

## Install

```
npm install asch-js
```

## Import

CommonJS

```
var AschJS = require('asch-js');
```

Front end

```
<script type="text/javascript" src="/browserify-asch.js"></script>

# 或者
<script type="text/javascript" src="/browserify-asch-min.js"></script>

# 访问
console.log(window.AschJS)
```



## Usage

Please reference the [asch http interface documents](https://github.com/sqfasd/asch_docs/blob/master/asch_http_interface.md)
